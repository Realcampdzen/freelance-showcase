# GitHub Repo Prototype (for Stepan)

Это прототип репозитория, который потом можно запушить на GitHub как базу личного бренда и фриланс-портфолио.

## Цель
- Хранить универсальное резюме
- Показывать кейсы и ссылки на деплои
- Быстро отправлять клиентам одну ссылку

## Структура
- `profile-readme/` — GitHub Profile README (для `username/username`)
- `portfolio/` — кейсы, стек, услуги, ссылки
- `assets/` — скриншоты, баннеры, превью
- `docs/` — процессы обновления и чеклисты
