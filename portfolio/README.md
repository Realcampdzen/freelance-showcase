# Portfolio

## Case 1 — RL Guide Book 2.0 (Camp platform)
- **Task:** Build a role-based gamified platform for camp operations and participant engagement.
- **Solution:** React + TypeScript frontend, cloud sync (Supabase), integrated backend parts (Node/Python), deployment on Vercel.
- **My contribution:** fullstack implementation + production stabilization.
- **Result:** working multi-role platform with iterative production fixes and stable release flow.
- **Tech:** React 19, TypeScript 6, Vite 8, Supabase, Vercel, Zustand.
